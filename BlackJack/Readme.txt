Pour démarrer le jeu:

	Il faut d’abord lancer le build du projet Cartes. Attention, lors du clone du projet on peut manquer le dossier lib. Si il n’y est pas créez le.

	Après avoir lancé le build de Cartes, lancez le build du projet blackjack.

	Plusieurs warnings vont s’afficher, ce sont des warnings sur la Javadoc, nous avons commenté toutes nos méthodes, mais pas les attributs car beaucoup ne servent qu'à afficher des composants Swing pour centrer d’autres éléments pour avoir une interface graphique de qualité.

Quelques informations diverses:

	Les statistiques forge sont faussées, Jonathan Raffin a beaucoup plus de changement que les autres personnes du groupe, car il a “ranger” les projets dans le dossier Livraison.

	Dans le jeu, dès que vous le démarrez, une partie se lance automatiquement. Pour redémarrer une nouvelle partie, il y a 2 possibilités:
		Soit la solde du Joueur est à 0 et du coup le jeu vous ramène sur le Menu mais relance une partie.

		Soit il faut fermer l’application et la relancer.

	Par manque de temps, suite à ce projet et aux projets des autres matières pour la licence, nous n’avons pas eu le temps de coder des tests unitaires. Nous avons quand même fait des tests manuels sur la totalité du jeu, des menus, des boutons, etc…
